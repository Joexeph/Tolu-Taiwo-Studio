🏞️ T.O.L.U T.A.I.W.O STUDIOS

“Capture Life’s Most Beautiful Moments.”
A professional photography website showcasing the work and services of T.O.L.U T.A.I.W.O Studios.
Built as a clean, responsive front-end site with HTML, CSS & JavaScript.

🌟 Features

📸 Multiple Photography Categories — Family Portrait, Product, Weddings, Lifestyle, Kids, etc.

📝 Service Information — “Learn more” sections for details about offerings.

📬 Contact / Booking Form — Direct way for clients to reach out.

📱 Responsive Design — Optimized for desktop, tablet, and mobile.

🎨 Visual-First Layout — Clean typography and imagery-focused design.


🛠️ Tech Stack

HTML5 – semantic structure

CSS3 – custom styling + responsive layout (Flexbox/Grid)

JavaScript (ES6) – interactivity and DOM manipulation


🔮 Future Improvements

✅ Image gallery/lightbox for better photo viewing

✅ Lazy-loading and compressed image formats (WebP)

✅ Accessibility enhancements (alt text, ARIA labels)

✅ SEO improvements (structured data, sitemap)

✅ Backend integration (Node.js/Express or headless CMS)

🧩 What I Learned

Structuring a real-world photography website using semantic HTML and modular CSS.

Building a fully responsive layout with Flexbox and CSS Grid.

Using JavaScript to enhance interactivity (menu toggles, form validation).

Deploying static websites seamlessly on Netlify.

The importance of optimizing images for fast load speeds.


🚧 Challenges

Handling large, high-quality photos without compromising page load speed.

Maintaining consistency across different devices and screen sizes.

Making the navigation intuitive while keeping the design minimalistic.

Learning deployment workflows (Netlify, GitHub Pages).
